Auth API Documentation

http://localhost:3000
POST /api/auth/register

Register a new user.

{
  "name": "Test User",
  "email": "taruna@yopmail.com",
  "password": "12345678"
}
POST /api/auth/login

Log in an existing user and receive a token.

{
  "email": "taruna@yopmail.com",
  "password": "12345678"
}
GET /api/auth/dashboard

Access a protected route. Requires a valid token.

Authorization: Bearer <token>
POST /api/auth/logout

Invalidate the session or token.

Authorization: Bearer <token>
POST /api/categories

Create a new category. JWT required

{
  "name": "Electronics",
  "description": "Electronic items and gadgets"
}
GET /api/categories

Get all categories.

PUT /api/categories/:id

Update a category. JWT required

{
  "name": "Updated Electronics",
  "description": "Updated description"
}
DELETE /api/categories/:id

Delete a category. JWT required

POST /api/products

Create a new product. JWT required

{
  "name": "iPhone 15 Pro",
  "description": "Apple's new flagship phone",
  "price": 1299.99,
  "stock": 50,
  "sku": "IPH15P-128GB",
  "imageUrl": "https://example.com/iphone15.jpg",
  "categoryId": "1"
}
GET /api/products

Get all products. Supports ?category=, ?search=, ?page=, ?limit= query params.

GET /api/products/:id

Get product by ID.

PUT /api/products/:id

Update a product. JWT required

{
  "price": 1199.99,
  "stock": 45
}
DELETE /api/products/:id

Delete a product. JWT required
